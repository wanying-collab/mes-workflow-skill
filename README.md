# MES Workflow Skill

AI workflow skill for MES worktime analysis and scheduling.

---

## Purpose

This skill is designed to analyze MES worktime records and generate scheduling suggestions.

---

## Features

- Worktime analysis
- Waiting time analysis
- Machine utilization
- KPI generation
- Lead Time calculation
- Scheduling suggestion

---

## Workflow

1. Read MES records
2. Normalize Start / Pause / Resume / End status
3. Calculate processing time
4. Calculate waiting time
5. Detect abnormal records
6. Generate KPI
7. Suggest scheduling
