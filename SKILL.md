# MES Workflow Skill

## Purpose

This skill is designed for MES worktime analysis and scheduling.

---

## Workflow

1. Read MES worktime records
2. Normalize Start / Pause / Resume / End status
3. Calculate processing time
4. Calculate waiting time
5. Detect abnormal records
6. Generate KPI
7. Suggest scheduling

---

## Features

- Worktime analysis
- Waiting time analysis
- Machine utilization
- Lead Time calculation
- Scheduling suggestion

---

## Outputs

- Total processing hours
- Waiting hours
- KPI dashboard
- Scheduling recommendation
